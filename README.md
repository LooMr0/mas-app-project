# vue-cli3-project 
#重庆元图移动端框架

## 安装依赖
```
npm install
```

### 运行  或者运行start.bat 脚本
```
npm run serve
```

### 打包命令
```
npm run build
```

### 运行测试
```
npm run test
```
### 框架说明
```
1、基础js库使用vue.js
2、移动端ui库cube-ui
3、框架地址：https://github.com/LooMr0/mas-app-project；
4、可以使用git工具clone到本地；
	git clone https://github.com/LooMr0/mas-app-project.git
5、cube-ui文档说明及使用手册参考：https://didi.github.io/cube-ui/

```
